# 2026-06-19 工作日志

## Huzoo（互圈）深化改造

### 后端新增功能
1. **DB 迁移机制**：`db.js` 加入幂等 migrate 函数，自动 ALTER TABLE 添加新字段
2. **城市字段**：`profiles.city TEXT`，支持城市筛选（`GET /profiles?city=北京`）
3. **微信号隐私**：未登录用户访问列表接口，`wechat` 返回「登录后可见」
4. **站内消息系统**：新建 `messages` 表 + `server/routes/messages.js`
   - `POST /api/messages`：发私信
   - `GET /api/messages/unread`：未读数
   - `GET /api/messages/threads`：会话列表
   - `GET /api/messages/thread/:userId`：对话记录（自动标记已读）
   - `DELETE /api/messages/:id`：删除消息
5. **`profiles.js`** 更新：`PUT /mine` 支持 city 字段，查询支持城市筛选
6. **server.js**：注册 `/api/messages` 路由

### 前端新增功能
1. **城市标签**：卡片头部展示城市，点击触发同城筛选
2. **城市筛选指示器**：激活筛选时顶部提示，点 ✕ 清除
3. **微信号保护**：`wechat-locked` 样式标灰未登录状态
4. **卡片详情弹窗**：点击卡片主体打开完整档案弹窗（offer 全量展示 + 收藏/私信按钮）
5. **发私信弹窗**：轻量 DM 弹窗，直接发送站内消息
6. **消息系统弹窗**：完整 inbox 界面，左侧会话列表 + 右侧聊天区，支持实时发送
7. **未读消息 badge**：顶栏「消息」按钮右上角红点，每 30 秒轮询
8. **卡片 offer 截断**：只显示前 3 条，超出显示「还有 N 条」
9. **城市字段表单**：档案填写/编辑表单新增城市输入框
10. **档案表单城市同步**：提交 PUT /mine 携带 city 字段
